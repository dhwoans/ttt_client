## 1. 턴 타이머

**상태**: 진행 중  
**파일**: `docs/TURN_TIMER.md`

각 플레이어가 제한된 시간 내에 이동해야 하는 핵심 게임 기능입니다.

- 🎯 **목적**: 게임 속도 조절 및 긴장감 조성
- ⏱️ **제한 시간**: 10초/턴
- 🎨 **시각 효과**: 3초 이하일 때 빨강색 강조
- 📊 **패널티**: 시간 초과 시 강제 패배

[자세히 보기](./TURN_TIMER.md)

## 2. 게임 모드

### 2.1 싱글 모드 (AI 대전)

- **경로**: `/game/single`
- **특징**: 로컬 AI와의 단독 게임
- **로직**: `src/features/game/single/`

### 2.2 멀티 모드 (온라인)

- **경로**: `/game/:roomId`
- **특징**: WebSocket 기반 실시간 대전
- **로직**: `src/features/game/multi/`

### 2.3 로컬 모드 (파티 게임)

- **경로**: `/game/:roomId?role=host|guest`
- **특징**: 호스트(큰 화면) + 게스트(컨트롤러)
- **로직**: `src/features/game/local/`
